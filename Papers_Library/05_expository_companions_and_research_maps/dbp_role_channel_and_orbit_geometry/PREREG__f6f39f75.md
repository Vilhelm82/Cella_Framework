```

h_ab = -(H_ab + H_{a,k} h_b + H_{b,k} h_a + H_{k,k} h_a h_b) / g_k      (a,b ∈ {i,j})
g_chart = (-h_i, -h_j, 1)
H_chart = [[-h_ii, -h_ij, 0], [-h_ij, -h_jj, 0], [0, 0, 0]]
```

`RoleChSpec(g, H) = { output role k -> Campaign A carrier of (g_chart, H_chart) : g_k ≠ 0 }`.

## Gauge solver

Given `g, H1, H2` (same `g`, `q≠0`): `Δ = H2 − H1`; require `Δ` symmetric; `a·g = gᵀΔg/(2q)`; `a = (Δg − (a·g) g)/q`; verify `Δ == g aᵀ + a gᵀ` exactly → `GAUGE_EQUIVALENT(a)` else `GAUGE_NOT_EQUIVALENT`. Different `g` → `GAUGE_NOT_COMPARABLE_DIFFERENT_G`. `GAUGE_UNDECIDED` only for a documented/typed singular limitation (not expected in the regular n=3 bound).
