# Archive — non-authoritative sediment

Nothing under this directory becomes current merely because it was once labelled
canonical, proven, certified, accepted, or complete.

| Directory | Contents |
|---|---|
| `pre-canon/portfolio/` | Former Lloyd portfolio entry point, standing results, corrections, registry, logs, and roadmaps |
| `Reference_Material/audits/` | Dated Cella catalogue and drift/audit material |
| `Reference_Material/v3_og/` | V3 and original-engine reference corpus |
| `superseded_docs/` | Documents already classified as superseded or scratch before retirement |
| `attic/` | Large historical reports, benchmark material, recovered code, and old work bundles |
| `handoffs/` | Historical session instructions and working-set snapshots |
| `packages/` | Zip bundles and exported skill packages |

The archive is preserved for provenance, comparison, and falsification. Use
`STATE/ARTIFACTS.json` for hashes and retrieval; use Cella's own admission process for
anything that may bear load.
