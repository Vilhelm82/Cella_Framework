## 1. Minimal DBP Curve versus Legendre Integration Curve

The minimal DBP curve recorded for the total-curvature constant is:

```text

y^2 = w*(1 - w)*(1 + w^2)
```

Equivalently, in minimal Weierstrass form:

```text
Y^2 = X^3 - X^2 + X - 1
```

```text
j(E_128) = 128
```

```text
Delta(E_128) = -256 = -2^8
```

However, the complete elliptic integrals in the closed forms are naturally written on a Legendre curve.

Define:

```text
lambda = (2 - sqrt(2)) / 4
```

```text
lambda_dual = (2 + sqrt(2)) / 4
```

Then:

```text
lambda_dual = 1 - lambda
```

```text

```

The Legendre curve is:

```text
E_lambda:
y^2 = (1 - x^2)*(1 - lambda*x^2)
```

or equivalently:

```text
Y^2 = X*(1 - X)*(1 - lambda*X)
```

For this Legendre curve:

```text
j(E_lambda) = 10976
```

and the exact modular-polynomial relation is:

```text
Phi_2(128, 10976) = 0
```

Therefore:

```text
E_lambda is 2-isogenous to E_128.
```

**Load-bearing correction.**

```text

- E_128: the minimal DBP j = 128 curve.
```

## 7. Pole Structure on the Elliptic Curve

On the affine `x`-line, the pole locations are:

```text

```

```text
x = -1/sqrt(n)
```

On the elliptic curve:

```text
y^2 = (1 - x^2)*(1 - m*x^2)
```

each `x` value has two sheets.

Thus there are four simple poles on the curve:

```text
(+x0, +y0)
(+x0, -y0)
(-x0, +y0)
(-x0, -y0)
```

The residues cancel pairwise. Depending on sheet convention, the residue list is one of:

```text

```

or:

```text

```

The invariant statement is:

```text

```

and:

```text
each pole pair over the same x-value has opposite residues.
```

This is the correct third-kind differential structure.
