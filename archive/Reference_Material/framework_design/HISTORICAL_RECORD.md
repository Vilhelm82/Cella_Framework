# V4 Historical Record

Scope: an AGNOSTIC, chronological record of V4 tasks and the decisions made, reconstructed from task numbers (task000..task050), dated report/handoff bodies, and the `_audit` arc windows. This file is the neutral spine — it reports "the record shows X" without imposing narrative, editorializing, or certifying scientific correctness. Git commit dates are unreliable (one snapshot lumps ~2 weeks), so ordering is by task number / dated body / arc window, with confidence cited per non-trivial claim. The companion file `TASKS_TO_REVISIT.md` carries the under-specced-task chain analysis (missed-context -> weaker-re-derivation -> scope-narrowing); the historical record itself stays neutral. Supersession is recorded ONLY where a later report genuinely replaced an earlier result — never on documentation-wording grounds.

Status legend: **PROVEN** (closed, evidence-backed) | **PARTIAL** (supported in scope but a sub-claim/mechanism unmet) | **REFUTED** (negative / hypothesis disproven) | **OPEN** (unconfirmed, declared-but-not-built, or explicitly unresolved; not re-verified here) | **SUPERSEDED** (genuinely replaced by a later artifact).

Context-flag legend (neutral, used only where the record shows it): `restart` (campaign restarted/renamed under a new name) | `weaker-re-derivation` (later work reached a weaker/partial result on ground earlier work closed) | `scope-narrowing` (acceptance/scope narrowed, sometimes post-hoc) | `pivot` (declared change of organizing principle) | `predates-prereads` (authored before the mandatory pre-reads existed, ~2026-05-24/26).
