**Status:** PROPOSED (Will ratifies rankings). Fresh authorship; prior results cited as
mathematics, reference only. Re-scored whenever a lever changes proof status.

## Scoring axes

- **Proof** — status of the load-bearing statement: PROVEN / CERTIFIED (exact, finite) /
  EMPIRICAL / OPEN.
- **Transfer** — distance from theorem to running sensor: 0 (is the sensor) → 3 (research
  programme away).
- **Blindness stmt** — does the lever ship a provable can't-see set? (Required for the
  credibility story.)
- **Invariance** — does it ship a provable can't-move-it group?
- **Prior-art** — proximity of adjacent literature: CLEAR / ADJACENT (named) / SWEEP OWED.
- **Reuse** — how many engine layers consume it.

**ROI = novelty potential + proof accessibility + structural depth + reuse − ambiguity.**

## Standing selection rule

The tell for a high lever is **derived impossibility + constructive completion** in one
object. Score candidates against that pattern first; levers with only elegance and no
blindness statement are presumption, not product. Re-rank at every proof-status change;
an OPEN → PROVEN move in Tier 2 usually outranks polishing anything in Tier 1.
