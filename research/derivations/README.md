# research/derivations — native derivation records

First-principles derivation trails. A result enters `STATE/RESULTS.md` (or a
formula enters `STATE/DEFINITIONS.md`) only with a derivation record or
certificate whose every step cites the corpus ids that license it.

Current records:
- `legendre_native.py` — machine-checked derivation of the E atom, the (K,E)
  connection, and the bilinear pinning invariant for the Legendre even quartic.
  Legendre's relation was NOT imported; it is derived from program primitives.
  Narrative writeup: `archive/pre-canon/E_ATOM_DERIVATION.md`. (The script imports
  a helper from a non-portable session path and is archived as a record, not run
  in-repo.)
