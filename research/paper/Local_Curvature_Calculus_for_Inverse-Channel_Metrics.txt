## Research Program: Local Curvature Calculus for Inverse-Channel Metrics

## Stage 1 — Define the object cleanly

The first job is to define the class of metrics you are studying.

A possible definition:

[
g=\operatorname{diag}(g_0,\ldots,g_m)
]

near a normal-crossing divisor

[
D={x_1\cdots x_k=0},
]

where each coefficient has a monomial-unit expansion

[
g_i
===

h_i(x,y)
x_1^{p_{i1}}\cdots x_k^{p_{ik}},
\qquad h_i(0,y)\neq 0.
]

Call this a **diagonal divisor-monomial metric germ**.

Then define the inverse-channel subclass by imposing the collapse pattern you care about, for example:

[
g_x\sim Bx^2,
\qquad
g_{y_\alpha}\sim A_\alpha x^{-2}
]

at a parity-fixed face.

The first deliverable should be a definition section with these notions:

[
\text{collapsing coordinate},
\quad
\text{transverse channel},
\quad
\text{metric valuation},
\quad
\text{parity-fixed germ},
\quad
\text{curvature valuation}.
]

The key mathematical object is:

[
\operatorname{val}_D(R[g]),
]

the pole order of scalar curvature along (D).

## Stage 2 — Prove the single-face theorems by hand

This should be my first serious proof target.

I already have two main local models.

### Theorem A: Generic quadratic collapse

Assume

[
g_x=Ax^2+O(x^3),
]

[
g_{y_\alpha}=P_{\alpha0}+P_{\alpha1}x+O(x^2).
]

Then prove

[
R=
\frac{1}{A}
\sum_{\alpha=1}^{m}
\frac{P_{\alpha1}}{P_{\alpha0}}
x^{-3}
+
O(x^{-2}).
]

This result explains the order-3 pole.

The conceptual message:

[
\boxed{
\text{Generic collapse detects transverse first drift.}
}
]

### Theorem B: Parity-fixed inverse-channel face

Assume

[
g_x=Bx^2+O(x^4),
]

[
g_{y_\alpha}=A_\alpha x^{-2}+O(1),
]

with all components even in (x). Then prove

[
R=
-\frac{m(m+5)}{B}x^{-4}
+
O(x^{-2}).
]

This is already in my note as the pure parity-fixed normal form, with the exact model

[
ds^2=Bx^2dx^2+x^{-2}\sum_{\alpha=1}^{m}A_\alpha dy_\alpha^2
]

giving

[
R=-\frac{m(m+5)}{B}x^{-4}.
]

For (m=2), this gives the Kerr–Newman coefficient

[
-\frac{14}{B}.
]

That is one of my strongest clean results.

## Stage 3 — Build the curvature-valuation formula

After the single-face proofs, derive a general formula for diagonal metrics.

For

[
g_i=e^{2u_i},
]

write scalar curvature in terms of (u_i), (\partial u_i), and (\partial^2u_i). Then plug in

[
u_i
===

\frac12
\left(
p_i\log x+\log h_i
\right).
]

This should expose why only the exponent data and leading units matter.

The target theorem:

> For a diagonal monomial-unit metric germ, the leading scalar-curvature pole is determined by the metric valuation vector ((p_i)), the leading units (h_i|_D), and finitely many derivatives of those units. Full expansion of (R) is unnecessary.

That theorem is the abstract version of my KN simplification.

## Stage 4 — Prove coordinate robustness

This is essential for credibility.

I need to know what happens under boundary-adapted coordinate changes:

[
x\mapsto c(y)x+O(x^2),
\qquad c(y)>0.
]

I do not need full coordinate invariance under arbitrary diffeomorphisms. I need invariance under the transformations that preserve the divisor structure.

Target theorem:

> The curvature pole order (\operatorname{ord}_D(R)) is invariant under boundary-adapted coordinate changes preserving the divisor filtration. In the parity-fixed case, the order-4 law is invariant under parity-preserving boundary changes.

This prevents the criticism:

> The pole order is a coordinate artifact.

I should separate three levels:

[
\text{pole order}
\quad
\text{leading coefficient}
\quad
\text{normal-form coefficient}.
]

The pole order should be invariant. The coefficient may transform unless normalized by a canonical defining function.

## Stage 5 — Develop the corner calculus

This is probably where my most original work lies.

For codimension two, take

[
D={xy=0},
]

and

[
g_i=h_i(s)x^{p_i}y^{q_i}.
]

The current note states that the polar support of (R) is controlled by vertices

[
V_1=(-(p_1+2),-q_1),
]

[
V_2=(-p_2,-(q_2+2)),
]

[
V_0=(-p_0,-q_0),
]

with explicit cancellation conditions.

Turn this into a full theorem.

The central theorem should look like:

> For a diagonal monomial metric near a codimension-two normal-crossing divisor, the leading scalar-curvature pole is the support function of a finite Newton polygon determined by the exponent matrix ((p_i,q_i)), after deleting vertices whose coefficients vanish.

Then define approach directions:

[
x=\rho\epsilon^a,
\qquad
y=\epsilon^b.
]

The order becomes

[
m(a,b)
======

\max_v -\langle (a,b),v\rangle.
]

This is powerful because it replaces two-variable Laurent chaos with convex geometry.

## Stage 6 — Generalize to codimension (k)

Do not try to fully solve this immediately. State it as a conjecture or partial theorem after codimension two is solid.

For

[
D={x_1\cdots x_k=0},
]

and

[
g_i=h_i x_1^{p_{i1}}\cdots x_k^{p_{ik}},
]

you want a finite set of curvature exponent vectors

[
\mathcal V(g)\subset \mathbb Z^k
]

such that along

[
x_j=\epsilon^{a_j},
]

the curvature pole order is

[
m(a)
====

\max_{v\in\mathcal V(g)}
-\langle a,v\rangle.
]

This would be the mature form of the theory:

[
\boxed{
\text{scalar-curvature asymptotics become tropical / Newton-polytope data.}
}
]

This is where you may connect cautiously to existing uses of Newton polytopes in scalar-curvature problems, such as Graev’s work on invariant Einstein metrics. But my setting is different: you are studying **local divisor valuations of diagonal metric germs**, not counting invariant Einstein metrics.

Relevant comparison:

*Graev, M. M. (2007). The number of invariant Einstein metrics on a homogeneous space, Newton polytopes and contractions of Lie algebras. Izvestiya: Mathematics, 71(2), 247–306.*
